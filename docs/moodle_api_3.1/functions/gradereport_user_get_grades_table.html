Get the user/s report grades table for a course


Arguments
courseid (Required)
        Course Id

General structure

int   //Course Id


XML-RPC (PHP structure)

[courseid] => int

REST (POST parameters)

courseid= int
userid (Default to "0")
        Return grades only for this user (optional)

General structure

int  Default to "0" //Return grades only for this user (optional)


XML-RPC (PHP structure)

[userid] => int

REST (POST parameters)
