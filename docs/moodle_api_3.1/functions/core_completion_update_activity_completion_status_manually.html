Update completion status for the current user in an activity, only for activities with manual tracking.


Arguments
cmid (Required)
        course module id

General structure

int   //course module id


XML-RPC (PHP structure)

[cmid] => int

REST (POST parameters)

cmid= int
completed (Required)
        activity completed or not

General structure

int   //activity completed or not


XML-RPC (PHP structure)

[completed] => int

REST (POST parameters)

completed= int


Response
General structure

object {
status int   //status, true if success
warnings  Optional //list of warnings
list of ( 
  //warning
object {
item string  Optional //item
itemid int  Optional //item id
warningcode string   //the warning code can be used by the client app to implement specific behaviour
message string   //untranslated english message to explain the warning
} 
)} 


XML-RPC (PHP structure)


    Array 
        (
        [status] => int        
        [warnings] =>
            Array 
                (
                [0] =>
                    Array 
                        (
                        [item] => string                        
                        [itemid] => int                        
                        [warningcode] => string                        
                        [message] => string                        
                        )
                )        
        )

REST

<?xml version="1.0" encoding="UTF-8" ?>
<RESPONSE>
    <SINGLE>
        <KEY name="status">
            <VALUE>int</VALUE>
        </KEY>
        <KEY name="warnings">
            <MULTIPLE>
                <SINGLE>
                    <KEY name="item">
                        <VALUE>string</VALUE>
                    </KEY>
                    <KEY name="itemid">
                        <VALUE>int</VALUE>
                    </KEY>
                    <KEY name="warningcode">
                        <VALUE>string</VALUE>
                    </KEY>
                    <KEY name="message">
                        <VALUE>string</VALUE>
                    </KEY>
                </SINGLE>
            </MULTIPLE>
        </KEY>
    </SINGLE>
</RESPONSE>
