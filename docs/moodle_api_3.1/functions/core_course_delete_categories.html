Delete course categories


Arguments
categories (Required)
        

General structure

list of ( 
object {
id int   //category id to delete
newparent int  Optional //the parent category to move the contents to, if specified
recursive int  Default to "0" //1: recursively delete all contents inside this
                                category, 0 (default): move contents to newparent or current parent category (except if parent is root)
} 
)
