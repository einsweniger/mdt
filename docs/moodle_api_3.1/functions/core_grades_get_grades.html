Returns student course total grade and grades for activities. This function does not return category or manual items. This function is suitable for managers or teachers not students.


Arguments
courseid (Required)
        id of course

General structure

int   //id of course


XML-RPC (PHP structure)

[courseid] => int

REST (POST parameters)

courseid= int
component (Default to "")
        A component, for example mod_forum or mod_quiz

General structure

string  Default to "" //A component, for example mod_forum or mod_quiz


XML-RPC (PHP structure)

[component] => string

REST (POST parameters)

activityid (Default to "null")

General structure

REST (POST parameters)
