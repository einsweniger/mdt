Return a list of notifications for the current session


Arguments
contextid (Required)
        Context ID

General structure

int   //Context ID


XML-RPC (PHP structure)

[contextid] => int

REST (POST parameters)

contextid= int


Response
General structure

list of ( 
object {
template string   //Name of the template
variables object {
message string   //HTML content of the Notification
extraclasses string   //Extra classes to provide to the tmeplate
announce string   //Whether to announce
closebutton string   //Whether to close
} 
} 
)

XML-RPC (PHP structure)


    Array 
        (
        [0] =>
            Array 
                (
                [template] => string                
                [variables] =>
                    Array 
                        (
                        [message] => string                        
                        [extraclasses] => string                        
                        [announce] => string                        
                        [closebutton] => string                        
                        )                
                )
        )

<?xml version="1.0" encoding="UTF-8" ?>

    <MULTIPLE>

                        <VALUE>string</VALUE>
                    </KEY>

                        <VALUE>string</VALUE>
